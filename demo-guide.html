ECS & ECR Quick Guide

Panduan praktis untuk mendepoy aplikasi Node.js menggunakan containerization ke infrastruktur AWS.

1
Persiapan Aplikasi

Pastikan Anda memiliki app.js dan Dockerfile sederhana yang sudah disiapkan di direktori proyek Anda.

Dockerfile 
FROM node:18-slim
WORKDIR /app
COPY package*.json ./
RUN npm install --production
COPY . .
EXPOSE 3000
CMD ["node", "app.js"]

2
Push Image ke ECR

Gunakan perintah berikut untuk mengirim Docker image lokal Anda ke Amazon ECR.

# 1. Login ke ECR
aws ecr get-login-password | docker login --username AWS --password-stdin [ACCOUNT_ID].dkr.ecr.[REGION].amazonaws.com

# 2. Build Image
docker build -t ecs-ecr-demo .

# 3. Tag & Push
docker tag ecs-ecr-demo:latest [ECR_URI]:latest
docker push [ECR_URI]:latest

3
Jalankan di ECS

Setelah image tersedia di ECR, Anda bisa membuat Task Definition dan Service di ECS Fargate.

Pro-tip: Gunakan Rolling Update agar aplikasi Anda tetap online saat ada pembaruan versi (no downtime).

# Paksa deployment ulang untuk mengambil image terbaru
aws ecs update-service --cluster ecs-lab-cluster \
    --service ecs-lab-service \
    --force-new-deployment
